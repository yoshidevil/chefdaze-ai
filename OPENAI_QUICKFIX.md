# 🚀 OpenAI Setup & Error Fix - Quick Card

## ⚠️ "Insufficient Quota" Error - QUICK FIX

### What Went Wrong?
Your free $5 credit expired OR no payment method added.

### Fix It Now (3-5 minutes)

**STEP 1:** Go here 👇
```
https://platform.openai.com/account/billing/overview
```

**STEP 2:** Do ONE of these:
- ✅ "Add payment method" → Add credit card
- ✅ "Set up paid account" → Add credit card  
- ✅ "Check plan" → Upgrade plan

**STEP 3:** Get new API key 👇
```
1. Go: https://platform.openai.com/api-keys
2. Click trash icon (delete old key)
3. Click "+ Create new secret key"
4. Copy immediately!
```

**STEP 4:** Use in app:
1. Open Recipe de SUCKERPUNCH
2. Sidebar → Select "Delicious de Recipe (OpenAI)"
3. Paste API key
4. See ✅ confirmation

**STEP 5:** Test it!
```
Ask: "I have chicken and lemon. What should I make?"
```

✅ **Done!** Should work now.
