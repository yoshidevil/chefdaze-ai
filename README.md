# 🍳 Cook de Staku – AI-Powered Smart Recipe Assistant

**Cook de Staku** is a modern, interactive cooking assistant built with **Streamlit**. It empowers home cooks of all skill levels to generate recipes, plan meals, substitute ingredients, and get smart culinary guidance.

With **12 unique AI personas**, each with a colored icon, your cooking journey becomes engaging, fun, and personalized. Whether you're a beginner or an experienced chef, **Cook de Staku** simplifies meal planning, recipe creation, and kitchen organization—all in a stylish, user-friendly interface.

---

## 🌟 Features

* **🍳 Recipe Generator**
  Enter your ingredients and receive a detailed, step-by-step recipe with cooking instructions, tips, and nutritional info.

* **🔄 Ingredient Substitute**
  Find alternatives for common ingredients like eggs, milk, or butter, ensuring your recipes never go wrong.

* **📅 Meal Planner**
  Generate a weekly meal plan with balanced and varied dishes to make your cooking schedule effortless.

* **🥗 Nutrition Calculator**
  View estimated calories, protein, carbohydrates, and fat per serving for your dishes.

* **🛒 Auto Grocery List Generator**
  Automatically create a shopping list based on your entered ingredients for stress-free grocery shopping.

* **📸 Recipe Images**
  Visual previews of your meals for inspiration and presentation.

## 🚀 Installation

Follow these steps to run **Cook de Staku** locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/cook-de-staku.git
cd cook-de-staku
```

### 2️⃣ Create a virtual environment (recommended)

```bash
python -m venv venv
```

Activate the virtual environment:

* **Windows**

```bash
venv\Scripts\activate
```

* **macOS / Linux**

```bash
source venv/bin/activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

> **Requirements** (in `requirements.txt`):

```
streamlit
```

### 4️⃣ Run the app

```bash
streamlit run app.py
```

### 5️⃣ Open in browser

Streamlit will display a local URL (usually `http://localhost:8501`) to interact with the app.

## 🧑‍🍳 Usage

1. **Select a mode** from the top navigation:

   * Recipe Generator
   * Ingredient Substitute
   * Meal Planner

2. **Input ingredients or questions** via the text box or chat interface.

3. **Receive smart AI responses**:

   * Recipes with step-by-step instructions
   * Nutrition estimates
   * Grocery lists

4. **Chat with the AI personas**: Each response comes from a randomly chosen persona with a colored icon for a fun and interactive experience.
